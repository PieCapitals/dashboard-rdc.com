## Usage

### Step 1

##### Config/config 

Change the following code with the localhost of your website, the config['baseurl'] must be the same like your own

--------------------------------------------------------------
```php
       $config['base_url'] = 'http://localhost:8888/dashboard-rdc.com/';
```
---------------------------------------------------------------


### Step 2 

##### Config/database.php
-------------------------------------------------------------------
```php
 
        $db['default'] = array(
	                            'dsn'	=> '',
	                            'hostname' => 'localhost',
	                            'username' => 'root',
	                            'password' => 'root',
	                            'database' => 'dashboard',
	                            'dbdriver' => 'mysqli',
	                            'dbprefix' => '',
	                            'pconnect' => FALSE,
	                            'db_debug' => (ENVIRONMENT !== 'production'),
	                            'cache_on' => FALSE,
	                            'cachedir' => '',
	                            'char_set' => 'utf8',
                            	    'dbcollat' => 'utf8_general_ci',
	                            'swap_pre' => '',
	                            'encrypt' => FALSE,
	                            'compress' => FALSE,
	                            'stricton' => FALSE,
	                            'failover' => array(),
	                            'save_queries' => TRUE
                            );

```
